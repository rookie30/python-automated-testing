# python-automated-testing
## 介绍

python自动化爬取51job中的python职位并存入excel

## 配置

需要安装chrome浏览器驱动：

https://chromedriver.storage.googleapis.com/index.html

找到对应版本号下载即可。



需要安装xlwt与selenium库：

```bash
$ pip install xlwt

$ pip install selenium
```

## 使用

需要将浏览器驱动`driver`的地址改为相应的地址。

需要将excel保存的地址改为相应的地址。
